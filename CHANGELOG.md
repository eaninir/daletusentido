# Changelog

## [v2.0] — 2026-05-27

### Agregado

* Nuevo diseño visual completamente renovado con identidad gráfica moderna y enfoque e-commerce.
* Hero slider personalizado con múltiples banners promocionales.
* Menú móvil responsive con botón hamburguesa.
* Implementación de modo oscuro mediante selector dinámico.
* Sistema de tarjetas deslizables para categorías destacadas.
* Botón flotante de WhatsApp con acceso directo a contacto.
* Optimización de imágenes utilizando formato WebP.
* Lazy loading en imágenes secundarias para mejorar rendimiento.
* Barra superior informativa con enlaces rápidos a redes sociales y ubicación.
* Navegación mejorada para catálogo y secciones internas.
* Nuevas animaciones y transiciones CSS.

### Mejorado

* Reestructuración completa del CSS utilizando variables (`:root`) y diseño más modular.
* Mejora significativa en responsive design para móviles y tablets.
* Optimización visual del header sticky.
* Mejora en accesibilidad visual y jerarquía tipográfica.
* Mejor organización semántica del HTML.
* Optimización de rendimiento general mediante reducción de peso en imágenes.
* Migración desde PNG hacia imágenes WebP.
* Mejora en experiencia de usuario (UX) y navegación.
* Formularios con mejor feedback visual usando Bootstrap Validation.

### Corregido

* Ajustes de visualización en pantallas pequeñas.
* Corrección de desbordamientos horizontales (`overflow-x`).
* Correcciones de alineación en tarjetas y sliders.
* Mejor manejo de espaciados y consistencia visual entre componentes.
